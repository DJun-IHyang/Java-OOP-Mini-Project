# Java OOP Mini Project
>Java OOP를 공부하면서 여러가지 이해가 덜된 부분과 내가 부족한 점을 발견했다.<br/>
1 - 클래스 다이어그램을 그리며 객체 간의 관계를 **시각화 하는 방법**
2 - SOLID 각각의 원칙이 왜 있는지는 알겠으나 자바 프로그래밍으로 생각해 봤을 때 각 메서드와 클래스가 어떤 방식으로 **호출**하고 **참조**하고 **반환**하는지 이해가 된 상태가 아니였다.
3 - 캡슐화, 상속, 다형성, 추상화등 핵심원칙을 눈으로만 보며 이해 하면 안된다 생각이 들었다.<br/>
그래서 나는 작은 프로젝트로 **RPG 게임의 캐릭터 생성**을
1 클래스 다이어그램을 만들어 시각화 하는 방법과 
2 핵심 원칙들(캡슐화, 상속, 다형성, 추상화, 인터페이스 등을 각 구간별로 나눠 사용해 보고
3 공부했던 Java 개념들을 머리속에 또한번 복습하는 것을 목표로 제작해보려 한다.
